# Justicia — Portfolio (Next.js Static Export)

This project is a Netlify-ready **Next.js** site (app router) exported as static files.

## What’s included
- Responsive portfolio with Hero, Services, Work, Blog, About, Contact
- Netlify Forms-compatible contact form
- Dark / Light mode toggle
- Instructions to deploy to **Netlify** and connect your custom domain
- Optional: instructions to wire a Notion CMS for the blog

## Quick start
1. Install Node 18+ and npm.
2. Copy the repo to GitHub or upload to Netlify directly.
3. `npm install`
4. `npm run build`
5. `npm run export`
6. Deploy the generated `out/` folder to Netlify (Netlify can do this automatically from your Git repo).
